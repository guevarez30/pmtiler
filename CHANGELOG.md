# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - Unreleased

- Initial public CLI for packing XYZ raster tiles into PMTiles v3 archives.
- Added raster rendering from GDAL-readable sources, including GeoTIFF, COG, and
  VRT mosaics.
- Added PNG, JPEG, and WebP tile output.
- Added archive inspection command.
- Added benchmark and quality scripts for the Blue Marble fixture.
